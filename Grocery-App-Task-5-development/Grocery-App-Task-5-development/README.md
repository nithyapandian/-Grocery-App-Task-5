* Frontend - reactjs
* Backend - Mockbee (Mock backend)

## Features

* Home Page
* Product Listing Page
* Filters by
  1. Price
  1. Category
  1. Ratings
* Cart Management
* Wishlist Management
* Authentication
  1. Sign-up
  1. Login
  1. Logout
